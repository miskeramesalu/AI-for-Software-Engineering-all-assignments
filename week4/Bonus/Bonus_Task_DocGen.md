### 2. DocGen AI Concept
DocGen AI (Documentation Generator AI) refers to the automated production of structured documentation—such as Markdown summaries, reports, and code annotations—based on generated or executed code. Using natural-language models, DocGen AI converts technical results into human-readable project deliverables.  
In this assignment, DocGen AI principles were used to:
- Automatically generate notes and structured markdowns (`Task1_Notes.md`–`Task3_Notes.md`).
- Summarize code objectives and results directly from execution.
- Enable reproducibility and transparency for evaluators.
