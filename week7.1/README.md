# AI Ethics Assignment

## Project Overview
This project contains the AI Ethics Assignment for the "AI for Software Engineering" specialization.  
It includes theoretical answers, case studies, ethical reflections, and practical bias/fairness audits using the COMPAS dataset.

The goal is to explore AI ethics principles, fairness, accountability, and transparency in AI systems.
